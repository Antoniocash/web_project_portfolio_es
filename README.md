# web_project_portfolio_es

ENGLISH /////

This project is part of the TripleTen curriculum and serves as a comprehensive competency test in modern interface development. The primary goal is to build a solid foundation that balances functional aesthetics with technical efficiency.

- Markup: Semantic HTML5 to enhance SEO and ensure the structure is screen-reader friendly (Accessibility/A11y).

- Styling: Advanced CSS3 utilizing:

  1.Flexbox for dynamic layouts and axis alignment.
  2.Positioning (Relative/Absolute) for precise control over overlapping elements.
  3.Responsive Design: Implementation of Media Queries for a seamless experience across mobile, tablet, and desktop.

- Version Control: Professional workflow using Git with deployment and documentation hosted on GitHub.

- The use of ChatGPT in this project was managed under a "Strategic Copilot" approach. Rather than blindly generating code, the tool was used to:

  1.Optimize resource and time management.
  2.Consult on specific syntax or problem-solving alternatives.
  3.Refine code architecture while maintaining a deep understanding and total control over every line implemented.

SPANISH /////

Portafolio para Desarrollador Web:

Este proyecto forma parte del currículo de TripleTen y funciona como una prueba integral de competencias en el desarrollo de interfaces modernas. El objetivo principal es construir una base sólida que equilibre la estética funcional con la eficiencia técnica.

- Maquetación: HTML5 semántico para mejorar el SEO y garantizar que la estructura sea comprensible para lectores de pantalla (Accesibilidad).

- Estilado: CSS3 avanzado utilizando:

  1.Flexbox para layouts dinámicos y alineación de ejes.
  2.Posicionamiento (Relative/Absolute) para el control preciso de elementos superpuestos.
  3.Diseño Responsivo: Implementación de Media Queries para una experiencia fluida en móviles, tablets y escritorio.

- Control de Versiones: Flujo de trabajo profesional con Git y despliegue/documentación en GitHub.

- El uso de ChatGPT en este proyecto se ha gestionado bajo un enfoque de "Copiloto Estratégico". En lugar de generar código a ciegas, la herramienta se utilizó para:
  1. Optimizar la administración de recursos y tiempo.
  2. Consultar sintaxis específica o alternativas de resolución de problemas.
  3. Refinar la arquitectura del código, manteniendo siempre un entendimiento profundo y control total sobre cada línea implementada.
